# キーボード入力

`kernel/io/keyboard.c`

## 仕様

| 項目 | 値 |
|------|-----|
| インタフェース | PS/2 キーボード（ポート `0x60`） |
| スキャンコードセット | Set 1（US 配列） |
| バッファサイズ | 256 文字（循環バッファ） |
| 割り込み | IRQ1（ベクタ 33） |

## API

```c
char keyboard_getchar();  // 入力があるまで hlt で待機してから 1 文字返す
```

## 対応キー

| キー | 文字 |
|------|------|
| `a`〜`z` | 小文字アルファベット |
| `0`〜`9` | 数字 |
| `Enter` | `\n` |
| `Backspace` | `\b` |
| `Tab` | `\t` |
| `Space` | ` ` |
| `-`, `=`, `[`, `]`, `;`, `'`, `` ` ``, `\`, `,`, `.`, `/` | 各記号 |

## 動作

- キーアップイベント（スキャンコード bit7 = 1）は無視する
- マッピングにない特殊キー（Shift, Ctrl, Alt, Fn など）は無視する
- Shift / CapsLock には未対応（全て小文字）
- `keyboard_getchar` は入力がない間 `hlt` で待機する（割り込みで起床）
