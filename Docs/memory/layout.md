# メモリレイアウト

## 物理メモリ

| 範囲 | 用途 |
|------|------|
| `0x0000`〜`0x04FF` | BIOS・リアルモード割り込みテーブル |
| `0x0500`〜`0x05FF` | E820 メモリマップ（24B × 最大 32 エントリ） |
| `0x7C00`〜`0x7DFF` | Stage1 MBR |
| `0x7E00`〜`0x9FFF` | Stage2 |
| `0x10000`〜`0x12FFF` | ブート時ページテーブル（PML4 / PDPT / PD） |
| `0x20000`〜 | カーネルフラットバイナリ |
| `0x100000`〜 | buddy アロケータ管理領域（最大 3MB/領域） |
| `0x200000` | ブート時カーネルスタック初期値（RSP） |

## 仮想アドレス空間

カーネルは `0xFFFF800000020000` にリンクされている（`linker.ld`）。

| 仮想アドレス | 物理アドレス | 用途 |
|-------------|------------|------|
| `0x0000000000000000`〜`0x3FFFFFFF` | 物理 0〜1GB | 恒等マッピング（PML4[0]） |
| `0xFFFF800000000000`〜`0xFFFF8000FFFFFFFF` | 物理 0〜1GB | Higher Half エイリアス（PML4[256]） |
| `0xFFFF800000020000`〜 | `0x20000`〜 | カーネルコード・データ |
